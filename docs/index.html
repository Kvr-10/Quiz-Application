Quiz Application

Interactive Quiz Game Developed in Java

About the Application

The Quiz Application is a Java-based game designed to test your knowledge through multiple-choice questions. Players can log in, read the rules, take the quiz, and view their scores at the end. The application resets the game state between sessions, allowing new players to start fresh.

For more details and to view the source code, visit:
GitHub repository

Features

  • User Authentication: Players can log in to start the quiz.
  • Rules Screen: Clear instructions provided before starting the quiz.
  • Interactive Quiz: Multiple-choice questions with immediate feedback.
  • Time Resets: Time reset after each question.
  • Score Display: Scores displayed at the end of the quiz session.
  • Reset Functionality: Game state resets between sessions for new players.

Skills

  • Java Programming: Core programming language for application development.
  • GUI Development: User interface designed using Java Swing.
  • Event Handling: Handles user actions and events within the game.
  • Data Persistence: Stores and retrieves user data during the quiz.

How to Run the Application

1. Clone the Repository

git clone https://github.com/Kvr-10/Quiz-Application.git

2. Navigate to the Project Directory

Quiz-App/out/artifacts/QuizApplication_jar/QuizApplication.jar

3. Run the Application

Locate the QuizApplication.jar file in the directory.

Double-click on QuizApplication.jar to start the game.
